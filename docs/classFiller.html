Detailed Description

template<class O>
class Filler< O >

static class for spinfull model term filler

Support:

  • Spinfull (deg = 4) aka Rashba type model
    • Spinfull::KineticTerm \(t_{ij}\)
    • Spinfull::RashbaXTerm \(\alpha_{ij}^x\)
    • Spinfull::RashbaYTerm \(\alpha_{ij}^y\)
    • Spinfull::RashbaZTerm \(\alpha_{ij}^z\)
    • Spinfull::ProxTerm \(\Delta_i \)
    • Spinfull::ZeemanXTerm \(V_i^X\)
    • Spinfull::ZeemanYTerm \(V_i^Y\)
    • Spinfull::ZeemanZTerm \( V_i^Z \)
    • Spinfull::ChemicalTerm \( \mu_i \)
  • Spinless (deg = 2) aka Kitaev type model
    • Spinless::KineticTerm \(t_{ij}\)
    • Spinless::ProxTerm \(\Delta_{ij}\exp(\phi_{ij}) \)
    • Spinless::ChemicalTerm \( \mu_i \)
